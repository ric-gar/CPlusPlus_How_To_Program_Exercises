Calculate the factorial of a nonnegative integer value


--First Refinement--

User enters a nonnegative integer value
Calculate factorial value
Print result


--Second Refinement--

Prompt user to enter nonnegative integer value
Declare integer variable (userInput)
Read input and give value to userInput

Initialize factorial value variable to 1 (factorialSum)

while userInput > 1
	factorial value = factorial value * userInput

	decrement userInput

Print factorial value
